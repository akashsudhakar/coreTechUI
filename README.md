## For Our Use
Run the following commands for deployment:
1. ng build --prod --base-href=/coreTechUI
2. open dist/index.html and change base href to "." It would be pointing to some location on your system
3. scp -r dist/* $TOMCAT_HOME/webapps/coreTechUI/

Depending on where you're deploying (dev/test) change the url-constants file before running the build and change the destination folder while deploying
